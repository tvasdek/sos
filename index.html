ΓΛΩΣΣΑ / LANGUAGE
ΒΗΜΑ 1 — TELEGRAM BOT TOKEN
Πώς να το πάρεις:
1. Άνοιξε Telegram → ψάξε @BotFather 2. Στείλε /newbot 3. Δώσε όνομα στο bot (π.χ. "Mama SOS") 4. Δώσε username που τελειώνει σε bot 5. Αντέγραψε το token που σου στέλνει
ΒΗΜΑ 2 — TELEGRAM GROUP CHAT ID
Πώς να το πάρεις:
1. Δημιούργησε group στο Telegram 2. Πρόσθεσε το bot σου ως Admin 3. Στείλε ένα μήνυμα στο group (π.χ. "test") 4. Άνοιξε browser και πήγαινε σε: https://api.telegram.org/botTOKEN/getUpdates 5. Βρες το "chat":{"id": — αρχίζει με -100...
